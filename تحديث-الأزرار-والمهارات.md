## 🎬 التأثيرات الجديدة

### الأزرار
```css
/* Ripple Effect */
.btn::before {
    content: '';
    position: absolute;
    width: 0;
    height: 0;
    border-radius: 50%;
    background: rgba(255, 255, 255, 0.2);
    transition: width 0.6s, height 0.6s;
}

.btn:hover::before {
    width: 300px;
    height: 300px;
}
```

```css
    transition: opacity 0.3s ease;
}

    opacity: 1;
}
```
